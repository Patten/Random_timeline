Module : Générateur de Slider

But du module :

Le module à pour but de générer un slider paramétrable depuis une administration wordpress. 

Fonctionnement :

Placer ce code <?php carousel_show();?>  a l'endroit voulu. 
Exemple dans le fichier header.php de votre thème.
Placer les fichier du plugin dans le fichier wp-content>plugin
Puis activer le plugin dans l'administration du wordpress

Pour crée les slide, aller sur l'onglet "Slides" puis sur "Ajouter". 
Enfin choisissez votre "image a la une" et le lien qui s'y rattache.

Options d'administrations

Sont paramétrables :

- la création et la modification des slides
- le lien clickable des slides
